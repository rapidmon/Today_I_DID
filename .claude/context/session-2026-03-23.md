## 주의사항

- **`android/` 폴더 주의**: `npx expo prebuild`로 덮어쓰면 위젯 코드가 다시 사라짐. 절대 `prebuild --clean` 재실행 금지
- **EAS Build 경로 문제**: Windows에서 `npx expo prebuild`하면 일부 설정에 Windows 절대 경로가 포함될 수 있음. EAS 클라우드는 Linux이므로 빌드 실패 원인이 됨
- **git push 금지**: 커밋까지만 하고 push는 사용자가 직접 수행
- **빌드/실행 명령은 사용자가 직접 실행**: Claude는 코드 수정만
