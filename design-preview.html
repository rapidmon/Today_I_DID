KEY CHANGES

1. 밝은 회색 배경 → 다크 네이비 (#0A0A1A)

2. 화이트 카드 → 반투명 다크 카드 + 미세한 보더

3. 두루마리 모달 → 네온 글로우 글래스 모달

4. 시스템 폰트 → Press Start 2P (제목) + Inter (본문)

5. 단색 골드 스코어 → 네온 옐로우 + 글로우 효과

6. 탭바 → 다크 배경 + 시안 네온 인디케이터

7. 체크박스 → 네온 시안 보더 + 완료 시 그린 글로우

8. 이모지 → 인라인 SVG 아이콘 (네온 글로우 + 스트로크 스타일)
